# Data Schemas
## Time-Place Celebrity Walkthrough Generator (TPCWG)

This document defines the key JSON data structures used in the TPCWG pipeline.

## 1. `job.json` (Input)
This file is the main configuration provided by the user to define the generation task.

### Schema
```json
{
  "location": "string",
  "venue": "string",
  "date": "string (ISO 8601 YYYY-MM-DD)",
  "num_subjects": "integer",
  "output_seconds": "integer",
  "era_style": "string",
  "camera_style": "string",
  "provider": {
    "llm": "string",
    "image_gen": "string",
    "video_gen": "string"
  },
  "subject_sources": "array[string] (optional)",
  "allow_living_people": "boolean (default: false, optional)",
  "allow_minors": "boolean (default: false, optional)",
  "labels": {
    "show_age": "boolean (optional)",
    "show_name": "boolean (optional)",
    "show_venue_titlecard": "boolean (optional)"
  }
}
```

### Example
```json
{
  "location": "New York City, NY, USA",
  "venue": "CBGB",
  "date": "1977-08-15",
  "num_subjects": 5,
  "output_seconds": 30,
  "era_style": "late-70s downtown punk club, gritty and raw",
  "camera_style": "handheld 16mm film, grainy, low light with lens flare",
  "provider": {
    "llm": "gemini-1.5-pro",
    "image_gen": "imagen-3",
    "video_gen": "veo"
  }
}
```

## 2. `subjects.json` (LLM Output)
This file is generated by the LLM during the refinement step. It serves as the final "cast list" for the video generation.

### Schema
```json
{
  "subjects": [
    {
      "name": "string",
      "birth_date": "string (YYYY-MM-DD) | null",
      "age": "integer | null",
      "evidence": "array[string]",
      "plausibility_note": "string",
      "prompt_image": "string",
      "prompt_video": "string"
    }
  ]
}
```

### Example
```json
{
  "subjects": [
    {
      "name": "Joey Ramone",
      "birth_date": "1951-05-19",
      "age": 26,
      "evidence": [
        "Associated with CBGB",
        "The Ramones were highly active in 1977",
        "Lived in New York City"
      ],
      "plausibility_note": "As the frontman for The Ramones, Joey Ramone was a central figure in the punk scene and a regular at CBGB during this era.",
      "prompt_image": "1977, inside a packed and sweaty CBGB club. The stage lights are low. In the foreground, a young Joey Ramone, tall and lanky with his signature sunglasses and leather jacket, watches a band. Captured on grainy, high-ISO 16mm film.",
      "prompt_video": "A short, handheld 16mm film clip. The camera pushes through a dense crowd at CBGB in 1977, finally landing on a close-up of Joey Ramone's face. He glances towards the camera before turning away. The scene is lit by dim, colored stage lights."
    }
  ]
}
```
